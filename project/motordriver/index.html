Project Details

  • 担当 : 回路班 / ソフト班
  • 時期 : 2018
  • 技術 : モーター制御 / 回路設計・発注 / はんだ付け

DCモータードライバ

ロボットの動力として多く用いられているモーター。モーターを任意の速度、方向で回転させるためには専用の電子回路が必要になります。市販もされていますが、市販品は高価で修理も困難です。そのため、回路班が設計したモータードライバ基板を使用しています。製作・修理・制御用ソフトウェア開発もサークル内で行っています。
